# Demo Script

1. 打开首页 `/`，说明这是供应商准入尽调 AI Agent 网站。
2. 点击“开始缓存演示”，进入 `/demo`。
3. 选择“比亚迪股份有限公司”案例，强调该流程不会消耗实时 API。
4. 进入 `/tasks/{task_id}`，先展示顶部完整 task_id 和数据对账卡片。
5. 展示联网搜索结果预览：真实 URL、query、rank、是否参与评分、不参与评分原因。
6. 展示企业画像字段：官网、行业、地区、注册资本、成立时间等，并说明需要官方工商核验。
7. 展示风险评分：比亚迪示例可评分风险证据为 0，普通记录不参与评分。
8. 打开报告区域，下载 Markdown 和 PDF，说明文件名包含当前 task_id。
9. 打开 `/settings/status`，说明密钥只在后端配置，前端只显示 masked 状态。

面试讲解重点：这个项目不是单次搜索工具，而是一个带证据链、消歧、评分、报告导出和部署模式的完整 Agent 产品闭环。
